# Go Toolbox

Репозиторий предназначен для накопления и систематизации шаблонов, примеров и переиспользуемых компонентов для проектов на Go.

Он служит рабочей базой для новых проектов и одновременно коллекцией решений для типовых инженерных задач, возникающих при разработке сервисов и приложений.

Репозиторий развивается как личный toolbox и справочник по архитектурным и инфраструктурным подходам.

## Принципы

- простота и читаемость,
- практическая применимость,
- модульность и переиспользуемость,
- ориентация на реальные проекты.

## Требования

Go 1.26+
